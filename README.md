## 项目安装
```
npm install
```

### 编译和启动 development 模式
```
npm run serve
```
or
```
npm run debug 
```
### 打包压缩文件
```
npm run pack
```
### 编译和构建生产版本
```
npm run build
```

## 在此基础上如何扩展?
目前的配置肯定还不能满足生产的需求，还需要扩展不少东西。  
使用vue-cli等就很方便对此进程扩展, 如安装各种vue插件等，还有使用原生模块，如'ffi'等, 但目前的配置是不能完成的，还需要以后多多完善。
